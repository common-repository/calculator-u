== Description ==

Simple and convenient calculator on the site. Simply enter the [uwp_calc] shortcode in a post or page.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPres.
1. Simply enter the [uwp_calc] shortcode in a post or page.
